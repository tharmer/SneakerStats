# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

### Contact
- Email: security@sneakerstats.example.com
- Response time: 48 hours

### Supported Versions
| Version | Supported |
|---------|-----------|
| 1.x     | Yes       |

### Scope
This policy covers all SneakerStats services and dependencies.
